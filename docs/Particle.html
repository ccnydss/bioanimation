Class: Particle

Particle(_center, _diam, _vel, _collidable, _color, _displayopt)

new Particle(_center, _diam, _vel, _collidable, _color, _displayopt)

Particles are the main class of the application. They possess a radius, color, position (maintained by their centers), and velocity. Every time a particle is drawn to the screen, its position vector is incremented by the velocity vector. There are 3 types of particles: Cl-, K+, and Na+ ions.
Parameters:
Name Type Attributes Default Description
_center Point The starting position of the new Particle
_diam integer The diameter of the particle
_vel Point The input velocity is represented as a Point object with (x, y) components.
_collidable boolean Determines if the particle will be impacted by collisions with Container objects.
_color Color | string A p5 Color object/string.
_display boolean <optional>
 true Determines if the particle object should be drawn to the screen at all.
