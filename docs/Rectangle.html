Class representing a Rectangle on the p5.js canvas.

Constructor

new Rectangle(_points, _fillColor, _borderColor)

Create a new Rectangle.
Parameters:
Name Type Description
_points Object An object of 4 points with keys named appropriately { _tl: p1, _tr: p2, _bl: p3, _br: p4}
_fillColor Color | string A p5 Color object, or a string that can be converted to a color (ex: "#ff0000"). Acts as the color for the inside area.
_borderColor Color | string A p5 Color object, or a string that can be converted to a color. Determines the border color, if a border is drawn.
