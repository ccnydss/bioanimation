All ions will inherit from the parent Particle class. Thus they can share methods and functionality with only cosmetic differences. The Na ion has its own properties listed below that are attached directly on the "Na" JavaScript class instead of each object instance. The point is to emulate static member variables, which JavaScript doesn't have.
